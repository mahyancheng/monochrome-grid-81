# HIDI LAU ARCHITECT — Official Website

**Live Site**: [https://hidilauarchitect.com](https://hidilauarchitect.com)

A portfolio and practice website for **HIDI Lau Architect** — a design-focused architectural firm based in Johor Bahru, Malaysia. The site showcases completed projects across residential, commercial, and hospitality sectors, and serves as the primary digital presence for the practice.

## Tech Stack

- [Vite](https://vitejs.dev/) — Build tool
- [TypeScript](https://www.typescriptlang.org/) — Type safety
- [React](https://react.dev/) — UI framework
- [shadcn/ui](https://ui.shadcn.com/) — Component library
- [Tailwind CSS](https://tailwindcss.com/) — Styling
- [react-helmet-async](https://github.com/staylor/react-helmet-async) — SEO management

The site uses **SSG (Static Site Generation)** via prerender — each route is exported as a standalone `index.html` file for optimal SEO performance.

## Local Development

Requirements: Node.js & npm — [install with nvm](https://github.com/nvm-sh/nvm#installing-and-updating)

```sh
# Step 1: Clone the repository
git clone <YOUR_GIT_URL>

# Step 2: Navigate to the project directory
cd <YOUR_PROJECT_NAME>

# Step 3: Install dependencies
npm i

# Step 4: Start the development server
npm run dev
```
